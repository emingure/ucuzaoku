# UCUZAOKU

Web scraping application to compare book prices through major Turkish book sellers(Kitapyurdu.com, dr.com, idefix.com, babil.com, nadirkitap.com)

## Usage
\<URL\>/\<query\> (https://ucuzaoku.herokuapp.com/elon%20musk)
